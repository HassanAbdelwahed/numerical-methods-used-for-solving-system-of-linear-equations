# numerical-methods-used-for-solving-system-of-linear-equations
## Objective
The aim of this project is to compare and analyze the behavior of the different 
numerical methods used for solving system of linear equations:
  1. Gauss Elimination.
  2. Gauss Jordan.
  3. LU Decomposition.
  4. Gauss Seidil.
  5. Jacobi Iteration.
